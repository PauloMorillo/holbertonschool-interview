# Infinite Multiplication

Create a script to do an infinite multiplication

## Requirements

gcc 4.8.4

## Run program

runnning and compiling C script:

```bash

gcc -Wall -pedantic -Werror -Wextra 0-mul.c _putchar.c -o mul

```


```bash

./mul 10 98

```
