# Holberton Interview

This repo has the exercise to prepare for an interview

## Requirements

Ubuntu(14.04)
gcc(4.8.4)

## Run program

first we should compile and later we can run or if you see a file with run permissions you can run it without to compile

```bash


gcc -Wall -Wextra -Werror -pedantic binary_tree_print.c 0-main.c 0-binary_tree_node.c -o 0-node

./0-node

gcc -Wall -Wextra -Werror -pedantic binary_tree_print.c 1-main.c 1-heap_insert.c 0-binary_tree_node.c -o 1-heap_insert

./1-heap_insert

```
