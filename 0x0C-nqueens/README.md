# Holberton Interview

The N queens puzzle is the challenge of placing N non-attacking queens on an NÃ—N chessboard. Write a program that solves the N queens problem.



## Requirements

Ubuntu(14.04)
python(3.4.3)
pep 8 (1.7.x)

## Run program

This is a python script for this reason the way to run is:

```bash

./0-nqueens.py 4

```
