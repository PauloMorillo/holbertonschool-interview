# Heap Sort

Write a function that sorts an array of integers in ascending order using the Heap sort algorithm

## Requirements


Ubuntu(14.04)
gcc 4.8.4

## Run program

GCC compiling code:

```bash

gcc -Wall -Wextra -Werror -pedantic 0-main.c 0-heap_sort.c print_array.c -o 0-heap

./0-heap

```
