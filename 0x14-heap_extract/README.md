# Heap Extract

Write a function that extracts the root node of a Max Binary Heap

## Requirements

gcc 4.8.4

## Run program

runnning and compiling C script:

```bash

gcc -Wall -Wextra -Werror -pedantic -o 0-heap_extract 0-main.c 0-heap_extract.c binary_tree_print.c -L. -lheap

```


```bash

./0-heap_extract

```
